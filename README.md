# 🖼️图像等比缩放器

输入:图像 (状态:可选)
输出:宽度/高度

宽度输入框: (默认:预设尺寸)
高度输入框: (默认:预设尺寸)

'预设尺寸'
320x704 (竖5:11)
384x640 (竖5:8)
448x576 (竖7:9)
512x512 (方1:1)
576x448 (横9:7)
640x384 (横8:5)
704x320 (横11:5)

获取宽高的优先级别顺序：
输入图像被连接，将获取源图像的宽高，强制关闭'预设尺寸'和宽度/高度的输入框。
输入图像未连接，则优先如下
1.‘预设尺寸’开启
2.宽度/高度的输入框 (‘预设尺寸’关闭)

# 📋图像尺寸选择器

输入:图像/遮罩 (遮罩状态:可选)
输出:图像/遮罩 (遮罩状态:可选)

宽度输入框:0 (默认:0)
高度输入框:0 (默认:0)

无论输入哪个宽度/高度的输入框，都按最大数字做缩放。
永久性保持比例的缩放图像。
